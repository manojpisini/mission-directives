# Visual Assets Batch Loop

Use `MD-192` to prove that custom code-native visuals materially improve the deliverable. Then use `MD-196` with `SKILL_ID visual-assets`, wrapped by `MD-197` and independently adjudicated by `MD-198`.

Outer queue: one item per infographic, vector, illustration, presentation asset, report exhibit, or animated explainer. Each item receives a `visual_asset_brief.schema.json` contract. The optional inner loop changes one failed criterion per pass and stops on verified success, plateau, budget, or safety.
